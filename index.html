Introducción
Vamos a empezar por el principio, Bootstrap es un framework CSS desarrollado inicialmente (en el año 2011) por Twitter que permite dar forma a un sitio web mediante librerías CSS que incluyen tipografías, botones, cuadros, menús y otros elementos que pueden ser utilizados en cualquier sitio web. Aunque el desarrollo del framework Bootstrap fue iniciado por Twitter, fue liberado bajo licencia MIT en el año 2011 y su desarrollo continua en un repositorio de GitHub.
Bootstrap es una excelente herramienta para crear interfaces de usuario limpias y totalmente adaptables a todo tipo de dispositivos y pantallas, sea cual sea su tamaño. Además, Bootstrap ofrece las herramientas necesarias para crear cualquier tipo de sitio web utilizando los estilos y elementos de sus librerías.
BootstrapCDN
- CSS
Para comenzar a implementar Bootstrap en tus proyectos, lo único que deber hacer es agregar a tu (head) el link de referencia de Bootstrap y sus estilos, el link se ve de la siguiente manera: link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/css/bootstrap.min.css" integrity="sha384-Vkoo8x4CGsO3+Hhxv8T/Q5PaXtkKtu6ug5TOeNV6gBiFeWPGFN9MuhOf23Q9Ifjh" crossorigin="anonymous"
Con esta simple linea de código ya tu sitio puede comenzar a adopar ciertas formas y estilos de Boostrap que te facilitarán tu proceso de creación
JavaScript
Para proyectos en los que desees cluir JavaScript, también deberás enlazar tu html a las funciones de JS que hayas creado, esto también se hace con un simple link, que trae Bootstrap; claramente la etiqueta "Script" debe ir cerrada adecuadamente
script src="https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/js/bootstrap.min.js" integrity="sha384-wfSDF2E50Y2D1uUdj0O3uMBJnjuUD4Ih7YwaYd1iqfktj0Uod8GCExl3Og8ifwB6" crossorigin="anonymous"
LayOut
En Bootstrap cada sección de información deve ir agrupada dentro de una etiqueta llamda "container", a esta se le agregarán diferentes caracteristicas o cualidades que harán a tu proyecto no solo más atractuvo visiualmente sino también tener un comportamiento Responsive. Tener un comportamiento Responsive quiere decir qye tu proyecto va a tener el mismo tipo de comportamiento y distribución sin importar desde qué tipo de dispositivos está siendo visitado o manipulado por los usuarios a los que está dirigido
Containers
Los contenedores son el elemento de diseño más básico en Bootstrap y son necesarios cuando se utiliza nuestro sistema de cuadrícula predeterminado. Los contenedores se utilizan para contener, rellenar y (a veces) centrar el contenido dentro de ellos.Estos contenedores, se escriben de la siguiente manera en tu código html:
  • All-in-one: class="container"
  • Fluid: class="container-fluid"
  • Responsive:
    class="container-sm" 100% wide until small breakpoint
    class="container-md">100% wide until medium breakpoint
    class="container-lg">100% wide until large breakpoint
    class="container-xl">100% wide until extra large breakpoint
Compatibilidad
Boostrap es compatible con la mayoría de navegadores web del mercado, y más desde la versión 3, actualmente es totalmente compatible con los siguientes navegadores:
  • Google Chrome (en todas las plataformas).
  • Safari (tanto en iOS como en Mac).
  • Mozilla Firefox (en Mac y en Windows).
  • Opera (en Windows y Mac).
  • Internet Explorer (en Windows y Windows Phone).
